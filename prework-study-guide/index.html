Prework Study Guide

HTML

  • One of the three cornerstone languages used in web development
  • Contains Text, Image, Header, & List elements
  • Builds the structure/Framework of a webpage
  • Also contains Meta-related, Content division and Semantic elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git checkout main - switch to main branch
  • git pull origin main - pull down latest branch
  • git checkout -b(Name) - creates a new branch
  • git status - tells what branch you are on
