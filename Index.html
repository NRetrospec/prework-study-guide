HTML

  • One of the three cornerstone languages used in web development
  • Contains Text, Image, Header, & List elements
  • Builds the structure/Framework of a webpage
  • Also contains Meta-related, Content division and Semantic elements

CSS

  • The Styling Property of a Webpage
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git checkout main - switch to main branch
  • git pull origin main - pull down latest branch
  • git checkout -b(Name) - creates a new branch
  • git status - tells what branch you are on

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a single variable that is used to hold a group of data.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • A string is a sequence of characters, typically used to represent text.
